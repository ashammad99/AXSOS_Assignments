# Simple Flask Routes App

This is a basic Flask project that demonstrates how to create routes, dynamic URLs, repeated messages, and a custom 404 error page.

## Project Features

- Home route
- Static route
- Dynamic route with a name
- Dynamic route with an integer and message
- Custom 404 error handler

## Requirements

- Python 3
- Flask

## Installation

First, activate your virtual environment.

Then install Flask:

```bash
pip install flask
