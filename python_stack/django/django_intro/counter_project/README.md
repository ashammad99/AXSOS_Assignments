# Session Counter (Django Project)

A simple Django web application that demonstrates how to use **sessions** to track user visits and maintain a counter value.

## Features

- Tracks number of page visits using Django session
- Maintains a session-based counter value
- Increment counter by fixed values (+1, +2)
- Custom increment using user input
- Reset session functionality

## Technologies Used

- Python 3
- Django
- HTML (basic templates)
- Django Sessions
